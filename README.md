# A.P.S---Acervo-de-Projetos-Senac
O Repositório PI Senac é uma plataforma para armazenar e organizar Projetos Integradores. Permite cadastrar projetos, registrar orientações de mentores e gerar relatórios para a coordenação. Também divulga eventos acadêmicos e disponibiliza metodologias de apoio ao desenvolvimento das equipes.
